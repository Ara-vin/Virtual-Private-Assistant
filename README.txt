How to run the project??
=======================

1. You will see a 'requirements.txt' which contains all the packages to run this project.
2. Install every package using command, (pip install packagename)
3. Create a new file to use as image database for face authentication and add its path to facerec.py
4. Run facerec.py to start with authentication. 
5. Run GUIASSISTANT.py to start without authentication.
